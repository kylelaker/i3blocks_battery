# i3blocks_battery

Displays an icon for the battery status as well as the current percentage as an
i3blocks block.

## Dependencies

 * cargo
 * rustc

## Building

    make

or

    cargo build

## Installing

    make install

will put the binary in `~/.local/bin/battery`.
